# Periodic tasks with CrateDB and Supertask

This folder includes a collection of examples about how to schedule periodic
tasks alongside CrateDB with [Supertask].

## Launch workload

Launch scheduled workload.
```shell
docker compose up --build scheduler
```

## User interface

Supertask is currently headless.


[Supertask]: https://supertask.readthedocs.io/
